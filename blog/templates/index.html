Welcome to TheBlog

TheBlog is a basic and simple Html and Django website with some few css

⁉ in the left side, there is a menu :
  • Click on blog to see posts (Published articles)
  • Click on Authors to see bloggers or authors with theirs articles and bio
  • Login is only for the admin

To run the project on your local,
On your terminal follow the below commands
git clone https://github.com/Theanayne/Basic-django-blog.git
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py collectstatic
python manage.py test
python manage.py createsuperuser
python manage.py runserver
